## Fonts

The following custom fonts are defined in the theme. They are loaded via the `/base.css` file.
To apply a font family or weight to an element you can use the tailwind utility classes.

```html
<div class="tw-font-sans tw-font-normal tw-italic" />
```

## Text styles

There are a couple of pre-defined text styles available. To apply a text style to an element you can use the tailwind utility classes.

```
<p class="tw-body-md" />
```
