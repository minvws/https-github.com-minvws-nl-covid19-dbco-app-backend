# Fundamentals - Shared styles

<br/><br/>

## Shared styles

Some styles are defined using [tailwind plugins](https://tailwindcss.com/docs/plugins) to make them easier for sharing across components.

### Basic text style

```
<p class="tw-body-md" />
```
